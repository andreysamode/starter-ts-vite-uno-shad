# React + TypeScript + Vite + UnoCSS + shadcn/ui

Andrey's starter project.

```
git clone https://github.com/andreysamode/starter-ts-vite-uno-shad .
npm i
vite
```

See App.tsx for examples of:
* UnoCSS Variant Groups transformer
* Coin flip animation (see uno.config.ts for keyframes)
* Radial gradients using UnoCSS syntax
* Google Fonts import
* shadcn/ui button (add new components with: npx shadcn-ui@latest add button)
